# objectron-demo

A demo tool for testing out the [Objectron](https://github.com/mena-devs/objectron) module

## Run local
```
npm start
```

## Generate a build 
```
npm run build
```
## Deploy build to demo page

```
git subtree push --prefix dist origin gh-pages 
```
